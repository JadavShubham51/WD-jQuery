JQuery Animation?
A-2 => we will see how to add or remove the CSS classes to an element using jQuery. To add the CSS classes to an element we use addClass() method, and to remove the CSS classes we use removeClass() method.
	
	AddCss() method

	RemoveCss() method
	sytanx:-
